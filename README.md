# Expense Tracker

## Description
Expense Tracker is a web application that helps users manage and track their daily expenses. Users can add, update, and delete expenses easily.

## Features
- Add Expenses
- Delete Expenses
- Update Expenses
- Track Daily Expenses
- User Friendly Interface

## Technologies Used
- Python
- HTML
- CSS
- JavaScript
- (Flask/Django agar use kiya hai to add karo)

## How to Run
1. Clone the repository
2. Open project folder
3. Run the main file
